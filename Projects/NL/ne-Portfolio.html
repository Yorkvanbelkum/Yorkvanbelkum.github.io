Eerste Website

Dit is de eerste website die ik heb gemaakt en ik denk dat het ook de website was waarvan ik het meest heb geleerd. ik was niet zo zeer gefocust op de front-end van de website maar meer op de coderen van de website en zorgen dat de website ook resposive zou zijn. in het begin was het erg moeilijk maar ik stopte veel tijd en inspanning in en toen begon ik al steeds meer dingen te leren.

Second Website

in de afbeelding hier boven zie je mijn 2de protfolio die ik gemaakt hebt sinds het begin van mijn cariere. ik heb heel veel dingen geleerd van dit project en was mezelf steeds mee bekent aan het maken met HTML en CSS. ik kwam dit project ook nog het parallax effect tegen. dus ben daar ook nog even mee bezig geweest terwijl ik deze website samen heb gesteld

Last Website

dit is de laatste Portfolio website die ik heb gemaakt, hier was ik meer aan het focussen op de front-end van de website. ik heb geprobeerd de website zo profasioneel mogelijk te maken zonder dat het te ingewikkeld werd.

Proces

ik was eerst begonnen een design bij mekaar te zetten in Adobe XD voordat ik begon met programeren omdat dan alles wat overzichtelijker zou zijn. ik kon het later altijd nog veranderen.

I first experimented in Adobe xd before I even started programming just so I knew what I wanted in the website. I always could change it later on.

Wat heb ik geleerd?

van het begin tot eind heb ik geleerd wat er allemaal mogelijk is in HTMl en CSS. ik heb de basics nu wel onder de knie. ik heb geleerd hoe ik mijn werk georganiseerd houd zodat niet alles door mekaar raakt. ik heb ook geleerd om met bootstrap te werken. Daar wil ik zeker nog wel meer in verdiepen.
